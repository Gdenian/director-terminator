Output: 7 个 handler 文件修改完毕，TypeScript 编译通过
</objective>

@src/lib/constants.ts — 当前 `getArtStylePrompt()` 的定义位置（即将废弃但保留供其他调用方）

<interfaces>
<!-- 新解析器签名（executor 需要知道调用方式） -->

resolveStylePrompt 位于 `@/lib/styles/style-resolver`，签名：
```typescript
export async function resolveStylePrompt(
  artStyle: string,
  userId: string,
  locale: 'zh' | 'en',
): Promise<string | null>
```

返回值可能是 null，调用方必须用 `?? ''` 提供默认值以保持向后兼容。

替换模式（per D-24, D-26）：
```typescript
// 旧（同步）：getArtStylePrompt(artStyle, locale)
const artStylePrompt = getArtStylePrompt(artStyle, job.data.locale)

// 新（异步）：await resolveStylePrompt(artStyle, userId, locale) ?? ''
const artStylePrompt = await resolveStylePrompt(artStyle, job.data.userId, job.data.locale) ?? ''
```

所有 Worker handler 都是 async 函数，可以直接使用 await。
</interfaces>
